HITTING
AVG
Batting Average — H / AB
OBP
On-Base % — (H+BB+HBP) / (AB+BB+HBP+SF)
SLG
Slugging % — Total Bases / AB
OPS
On-Base + Slugging — OBP + SLG
wOBAADV
Weighted On-Base Avg — weights each outcome by run value (BB×.69, 1B×.88, 2B×1.24, 3B×1.56, HR×2.00)
wRC+ADV
Weighted Runs Created+ — league-adjusted wRC, 100 = average, >100 above average. Regressed toward 100 for <150 PA.
oWARADV
Offensive WAR — wins above replacement from batting. Based on CCAA run environment (13.6 runs/win). Regressed for <200 PA.
BB/KADV
Walk-to-Strikeout Ratio — BB / K. Plate discipline metric; higher = better.
PITCHING
ERA
Earned Run Average — (ER × 9) / IP. CCAA league avg: 4.61.
WHIP
Walks + Hits per Inning Pitched — (BB+H) / IP
K/9ADV
Strikeouts per 9 innings — (K × 9) / IP. CCAA league avg: 8.3.
BB/9
Walks per 9 innings — (BB × 9) / IP. CCAA league avg: 4.9.
K/BBADV
Strikeout-to-Walk Ratio — K / BB. Command metric; higher = better.
K%ADV
Strikeout Rate — K / Batters Faced. More reliable than K/9 for small samples.
ERA+ADV
ERA adjusted to CCAA average — (lgERA / ERA) × 100. 100 = avg, >100 is better. Regressed for <40 IP.
pWARADV
Pitching WAR — wins above replacement based on ERA vs CCAA average. Regressed for <50 IP.
★ All WAR values use CCAA run environment: lg ERA 4.61, runs/win 13.6, replacement level −33.4 runs/600 PA. Small-sample regression applied: wRC+ and ERA+ converge toward 100 (average) below threshold PAs/IP.
